<h1>pyfetch</h1>
Python utility that shows your system information. Requires Python3+. Working only on Linux systems now.
<h2>Install & Build</h2>
<code>git clone https://github.com/wr4ck3r/pyfetch.git && cd pyfetch && ./build.sh</code>
